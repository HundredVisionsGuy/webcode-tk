Font-size Experiment

An experiment for testing computed font sizes.

General Notes

  • The base font-size by default is 16px
  • I set the body element to 20px to make font-size calculations a little simpler

Keywords

Absolute Keywords

Absolute keywords are always set in reference to the root font size. Each size in the list below will have that size applied.

  • xx-small
  • x-small
  • small
  • medium
  • large
  • x-large
  • xx-large
  • xxx-large

Relative Size Keywords

There are just two relative size keywords: smaller and larger. These bump up or down based on the computed font-size

The keyword larger makes the text 120% larger than whatever the computed font would be.

The keyword smaller makes the text 83.3% smaller than the computed font. Below are some examples.
