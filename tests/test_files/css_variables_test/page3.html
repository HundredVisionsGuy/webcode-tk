Structure

Below is the structure of CSS for Page 1:

  • reset.css - defines --primary-color: #333 on :root
  • theme.css - usescolor: var(--primary-color) on p
  • <style> tag - redefines -primary-color: #666 and uses it in spite of it's demonic hex code.

This tests: variable definition in first stylesheet, usage in later stylesheet, and override in style tag.

NOTE: page 2 uses the same two external stylesheets, but a different style tag.
